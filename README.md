# 資格試験成績管理システム

## 概要

Javaの学習成果として開発した資格試験成績管理システムです。

資格試験ごとの得点や学習時間を記録し、学習状況を可視化することを目的として開発しました。過去の試験結果を保存し、平均点や累計学習時間を確認できます。

---

## 開発背景

Java Silverの学習を進める中で、資格試験の勉強状況を管理できるシステムがあれば便利だと考え、本システムを開発しました。

本システムの開発を通して以下の技術の理解を深めることを目的としています。

- オブジェクト指向設計
- Java SwingによるGUI開発
- コレクションフレームワーク
- ファイル入出力
- データ永続化

---

## 主な機能

### 1. 資格登録機能

新しい資格試験を登録できます。

例

- Java Silver
- 基本情報技術者試験
- 応用情報技術者試験

---

### 2. 成績登録機能

資格ごとに以下の情報を登録できます。

- 得点
- 学習時間
- 受験日

---

### 3. 成績分析機能

登録された成績から以下を算出します。

- 平均点
- 最高点
- 最低点
- 累計学習時間

---

### 4. 検索機能

資格名や説明文から資格を検索できます。

---

### 5. データ保存機能

登録したデータをファイルへ保存できます。

保存ファイル

```text
data/exams.txt
data/results.txt
