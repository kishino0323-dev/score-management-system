# 資格試験成績管理システム

## 概要

資格試験の学習状況を管理するために開発したJavaアプリケーションです。

資格ごとの得点や学習時間を記録し、平均点や累計学習時間を確認できます。
データはファイルへ保存され、アプリを再起動しても復元できます。

---

## 開発背景

Java Silverの学習を進める中で、自身の資格試験学習の進捗を記録・管理できるシステムが欲しいと考えました。また、学校で習ったことを活かしてJPanelで何か作れないかと思い開発しました。

本システムの開発を通して、

- オブジェクト指向設計
- Java SwingによるGUI開発
- ファイル入出力
- コレクションフレームワーク

について学習しました。

## 主な機能

### 資格登録

新しい資格試験を登録できます。

### 成績登録

資格ごとに以下を記録できます。

- 得点
- 学習時間
- 受験日

### 学習状況確認

以下の情報を確認できます。

- 平均点
- 累計学習時間

### データ保存

資格情報と成績情報をファイルに保存します。

### データ読込

起動時に保存済みデータを読み込みます。

## 工夫した点

### CardLayoutによる画面切り替え

機能ごとに画面を分割し、操作しやすいUIを実現しました。

### データの永続化

FileDataManagerクラスを作成し、資格情報と成績情報をテキストファイルへ保存できるようにしました。

### 責務の分離

- Exam：資格情報管理
- ExamResult：成績情報管理
- ExamManager：ビジネスロジック
- FileDataManager：ファイル操作
- ScoreManagementUI：GUI

としてクラスごとの役割を分離しました。
